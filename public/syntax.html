Basic Syntax

ForTemp uses ES6/ES2015 Template Literals
Normal Template Literals look like this `Hello ${name}!!!`
Let's Break This down!!!

  • `` - the back ticks mark a template literal (ForTem handles this for you)
  • ${name} - the text inside the ${} is the name of a variable to put in the string
  • Hello ${name}!!! - everyting outside the ${} is returned along the new text in the output

Syntax Examples

Template

Hello ${name}!

Data

name = "World"

Hello World!
